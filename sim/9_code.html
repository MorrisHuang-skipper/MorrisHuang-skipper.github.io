The following code is about how to simulate the eletric field and potential of a dipole.
You can also view the complete code here(full code).

--> First you need to import vpython.
--> Then setting some important parameter.
--> Now come to the main function which will be execute at the end.
--> Main function is pretty straight forword, first create a scene that set userspin, userpan and userzoom to False that desable scene to spin, pan and zoom.
--> Then create a dipole that use the class 'Charge' that will explain later.
--> Finally create field and potnetial using function in the class 'Charge'.

--> Create field in the given range and set the default interval be the radius of the charge.
--> Using for loop that create arrow wihtin the meshgrid.
--> Using try-except syntax that prevent the error occured at the surface of the charge.
--> Use function field_color that determine the color of the arrow at the given position that represent the magnitude of the eletric field.

--> Using for loop that create quad wihtin the meshgrid.
--> Using try-except syntax that prevent the error occured at the surface of the charge.

--> By using above normalized value the following two function that use these value to create a vector of color that deternine the color of arrow and quad object that representing the strenght of field and potential.
--> Using vpython object 'quad' that takes four vertex as input and create a quad with unique color that represent eletirc potential strength.
